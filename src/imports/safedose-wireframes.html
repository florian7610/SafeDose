L3
Component Diagram
Feature focus: Automated Drug Interaction Checker (GET /api/interactions)
🔍 Feature: Automated Interaction Checker — Node.js/Express API Server

Auth Middleware

Validates JWT from HTTP-only cookie on every request. Attaches req.userId to the request context. Rejects unauthenticated calls with 401.

Backend

Interaction Controller

Handles GET /api/interactions. Orchestrates: fetch user meds → extract RxCUIs → call FDA service → parse results → return warnings array.

Backend

Medication Repository

Mongoose model layer. Queries medications collection filtered by user_id. Returns list of {drug_name, rxcui} pairs.

MongoDB

openFDA Service

HTTP client wrapper. Calls api.fda.gov/drug/label.json?search=openfda.rxcui:XXXX. Extracts drug_interactions field. Caches results in memory for 24h.

External API

Interaction Parser

Pure function. Takes raw FDA text responses, cross-references drug pairs from the user list, classifies severity (HIGH/MODERATE), extracts warnings for display.

Backend Logic

InteractionAlert Component

React component. Receives parsed warnings array as props. Renders color-coded alert cards, severity badges, and FDA source links on the Safety Center page.
